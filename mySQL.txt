DCL (Data Control Language)
Grant revoke
    GRANT SELECT, INSERT ON mydb.* TO 'user1'@'localhost';
    REVOKE INSERT ON mydb.* FROM 'user1'@'localhost';
TCL (Transaction Control Language)
START TRANSACTION COMMIT ROLLBACK SAVEPOINT
    START TRANSACTION;
    UPDATE accounts SET balance = balance - 100 WHERE id=1;
    UPDATE accounts SET balance = balance + 100 WHERE id=2;
    COMMIT;

    SAVEPOINT sp1;
    ROLLBACK TO sp1;

🔥 GROUP BY + HAVING
SELECT user_id, COUNT(*) 
FROM orders
GROUP BY user_id
HAVING COUNT(*) > 5;

HAVING = aggregate filter
WHERE = row filter

🔥 Interview Question:

Second highest salary find karo

Method 1:

SELECT MAX(salary)
FROM employees
WHERE salary < (SELECT MAX(salary) FROM employees);

Method 2 (Better):

SELECT salary
FROM employees
ORDER BY salary DESC
LIMIT 1 OFFSET 1;
🪟 Bonus: Window Functions (Advanced Level)

SELECT 5 << 1;  -- 10 means mult by 2
SELECT 8 >> 1;  -- 4 means divide by 2
🔥 Real Production Use Case: Feature Flags

Suppose:

Permission	Value
READ	1
WRITE	2
DELETE	4
ADMIN	8

User permissions = 5

Binary:
5 → 0101
Means READ (1) + DELETE (4)

Check permission:

WHERE (permissions & 4) != 0

🔥 Ye production-grade answer hai.

🔥 Interview Question
Why use bitwise instead of boolean columns?

Answer:

“Multiple boolean columns ke jagah single integer me compact permission store kar sakte hain, memory efficient hota hai aur faster comparison hota hai.”
